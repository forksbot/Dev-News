# News

## Installation
1. Go to [snacky-news-rs](https://github.com/huytd/snacky-news-rs) clone and run 
2. Clone this repo `https://github.com/ldakhoa/Dev-News.git`
3. cd /Dev\ News/
4. `pod install`
5. Build and run it
